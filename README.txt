Берём на 50% VXX, на 50% TQQQ.
Для TQQQ чек в конце месяца по SMA_200_SPY. Рейтио 1.4, стоп 0.7. 
Для VXXB рейтио 0.91, стоп 0.3. 
Доходность 33.7%. 
Предельные просадки 21%.
Если добавить бонды, доходность вырастет на пяток процентов. 

На текущий момент из агрессии есть эти плечевые ETF и еарнинги. Можно 75% отправить в плечевые (VXXB требует > 100% маржи, однако TQQQ < 100%), а 25% в еарнинги (на еарнинги 
маржа 25%, таким образом на 25% капитала можно грузить 100% порта).
Или 50% на 50%, если требуются еарнингам вторые плечи. 

Например, еарнинги дают 30% годовых, с просадкой 6%. 
Значит при загрузке 75%, на 25% порт будет приносить:
(33.7% * 0.75) + (30.0 * 1.0) = 55.3
Просадки:
(21 * 0.75) + *6 * 1.0) = 21.75%.
